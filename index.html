Abstract

GANs (Generative Adversarial Networks) is a hot topic in the Deep Learning research community. Some of the main application areas of GANs are image generation, text generation, translation and more. In this project, we use GAN to learn cross domain relationships between data from two domains. In this work (and also in the the work described in the paper which is ur main reference), we do not use pair-pair data for training. Instead we have a set of images from domain 1 and a set of images from domain 2. We then try to learn the relation between the domains. Given a new imag ein domain 1,the model will generate a matching image from domain 2.
